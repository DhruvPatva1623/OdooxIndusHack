# Security Policy

## Security Policy Overview
This document outlines the security policies for the OdooxIndusHack repository by DhruvPatva1623. We are committed to maintaining the security of our code and providing a secure experience for our users.

## Reporting a Vulnerability
If you discover a security vulnerability within this repository, please follow the instructions below to report it:
1. **Do Not Share**: Avoid public disclosure of the vulnerability until it has been addressed.
2. **Create an Issue**: Please open an issue in this repository, including detailed steps to reproduce the vulnerability.
3. **Confidentiality**: We will treat your report with strict confidentiality and will work to resolve the issue as quickly as possible.

## Response Time
We aim to respond to all security vulnerability reports within 48 hours. If you have not received a response after 48 hours, please follow up via the issue or contact the repository maintainers directly.

## Acknowledgements
We thank you for helping to keep our project secure! Any contributions to the security of this project will be acknowledged and appreciated.
